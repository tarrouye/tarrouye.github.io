Computer Networks

I took a course and lab in Computer Networking. In this course we studied the following:

Internet and the building blocks of computer networks and their respective OSI and TCP/IP models, LAN, WAN, Broadband, and Wireless.

Services in the application layer, including network tools, DNS, email, and the web as well as client-service and peer-to-peer applications.

System calls and socket programming tools in Unix/Linux and C to build transport layer TCP/IP and UDP/IP reliable data transfer services with flow and congestion control.

Network layer in the context of both per-router and SDN settings and select and use appropriate techniques for IP addressing, subnetting, SNMP, ICMP, DHCP, NAT, and interior and exterior routing in the network layer, including LS, DV, RIP, OSPF and BGP protocols.

Techniques for error control, flow control, ARP and MAC addressing, Ethernet and VLAN, and multiple access control protocols: channel partitioning (TDMA, FDMA, CDMA), random access (Aloha, CSMA, CSMA/CD, CSMA/CA), and taking turns (polling, token passing).

Wireless networks (SNR, BER, 802.11 LAN architecture, channels, and multiple access protocol), cellular networks (2G, 3G, 4G, LTE), mobile IP, and GSM

Computer network security, including cryptography (DES/AES, RSA, MD5/SHA2), end-point authentication (public key infrastructure, digital signatures), and securing the Internet (Application layer security, Transport layer security (TCP/SSL), Network layer security (IPsec and VPN, and wireless network security.

Operating Systems

I took a course and lab covering Operating Systems. In this course we studied the following:

Basic OS functionality: process management, main memory management, secondary storage management, I/O devices, file management, security.

Process management, such as allocating resources to processes, CPU scheduling, handling inter-process communication, multi-threading, synchronization, deadlock detection and avoidance.

Address translation and memory management, such as base and bound, segmentation, paging, paging with TLB, and multi-level translation

Caches and virtual memory management and page replacement algorithms, such as FIFO, LRU, LFU, MIN, and Clock

File system management, such as directory structure, layout management, partitioning and mounting, buffering, and data space management.

Storage devices, physical memory map, storage structure, performance, storage array, SAN, and NAS

I/O management and disk scheduling algorithms, such as FCFS, SSTF, SCAN, C-SCAN, LOOK, and C-LOOK

Implementation of schedulers, memory managers, and file-systems algorithms. Modify and extend an operating system such as MINIX.
